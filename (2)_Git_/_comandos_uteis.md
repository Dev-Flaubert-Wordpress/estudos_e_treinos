# Git & GitHub
Comandos essenciais:

1. ``sh git status ``
2. ``sh git add --all ``
3. ``sh git commit -m "branch Descricao da tarefa" ``
4. ``sh git push origin branch ``
5. ``sh git branch ``
6. ``sh git branch ... ``
7. ``sh git checkout branch ``
8. ``sh git merge branch -m "Mescla com a branch ..." ``
